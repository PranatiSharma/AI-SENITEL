# AI-SENITEL
Sentinel Engine is a real-time cybersecurity system that detects malicious URLs using automated threat intelligence. Built with a Python Flask backend, validates requests against threat datasets. Using Pandas for efficient data analysis, identifies phishing and cryptojacking patterns, ensuring fast, secure, and reliable detection via RESTful APIs.
